# After Dusk

# Game Design Document

## Project Vision

After Dusk — хардкорная top-down игра про выживание в огромном процедурно генерируемом мире после зомби-апокалипсиса. Главная идея игры — предоставить игроку полную свободу действий. Здесь нет сюжетной кампании, обязательных заданий и линейного прохождения. Игрок сам выбирает, как выживать: путешествовать, укреплять найденный дом, строить лагерь, ремонтировать транспорт или исследовать мир.

## Основные принципы
- Огромный мир.
- Реалистичное выживание без лишнего реализма.
- Исследование всегда вознаграждается.
- Любой предмет имеет применение.
- Шум является одной из важнейших механик.
- Игрок постоянно принимает решения и рискует.

## Игровой цикл
Исследование → Поиск ресурсов → Выживание → Улучшение снаряжения → Укрепление убежища → Экспедиции → Ремонт транспорта → Повторение.

## Цель игры
Конечной цели нет. Игрок самостоятельно определяет свой стиль игры и старается прожить как можно дольше.

# 1. Жанр
Hardcore Survival Sandbox, Top-Down, Open World, RPG, Coop.

# 2. Основа мира
Игрок выбирает размер мира при создании. Один тайл равен одному метру. Мир состоит из чанков 32×32 тайла. Генерируются дороги, города, деревни, леса, поля, озёра, реки, болота, магазины, склады, больницы, полицейские участки, фермы, заправки, военные объекты и уникальные точки интереса.

# 3. Базы
Игрок не строит полноценные дома. Вместо этого он занимает существующие здания, укрепляет их заборами, шипами, баррикадами и простой электроникой. Для временного проживания доступны палатки и тенты.
